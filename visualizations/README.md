# Groupie tracker - Visualizations

Визуализация данных Groupie tracker построена на принципах "Schneiderman's 8 Golden Rules" и best-practice UI/UX: все данные отображаются понятно, быстро, без лишней нагрузки на пользователя.

## 📝 Описание

### 🎨 Контраст цветов:
- в светлой теме текст — #1C2233 на #F0F4FF;
- тёмной — #E5E8F0 на #111620;
- фон - используется нейтральный для обоих режимов (light/dark), без агрессивных цветовых переходов;
- для интерактивных элементов (кнопки, ссылки) используются цвета с хорошим контрастом и hover-эффектами.

### 🔄 Консистентность дизайна:
- единая цветовая палитра для всех страниц;
- выравнивание и сетка выдержаны для карточек, списков, подробных карточек;
- управляющие элементы (кнопка обновить, switch темы) всегда в одной зоне (top-bar).

### 🖥️ Адаптивность:
- верстка на flex, карточки сжимаются до min-width;
- минимальная ширина защищает от перелома дизайна;
- все основные элементы читаемы даже при уменьшении окна браузера;
- информативные обратные связи (feedback, error-handling);
- поддерживает быстрые действия и легкий возврат/отмену.

### 👆 Удобство интерфейса:
- все действия пользователя доступны за 1–2 клика;
- фокус на поиск и навигацию;
- нет скрытых меню или непредсказуемых реакций.

### ⚠️ HTTP-errors:
- фирменный шаблон ошибок используется для всех ошибок (400/404/500) — нет "сырых" ответов Go;
- любой несуществующий маршрут (например, /abracadabra) вызывает стилизованную страницу 404 с понятным сообщением;
- используется тот же фирменный шаблон (error.html);
- сообщение об ошибке и номер кода видны на странице.

## 🌙 Дополнительные визуальные и UX-фишки:
- cмена темы (светлая/тёмная), с автоматическим сохранением в localStorage;
- адаптивный top-bar (логотип, кнопка обновить, switch темы);
- кеширование данных и статики для мгновенной загрузки;
- SVG-заглушки для битых картинок;
- минималистичная структура кода — весь UI разбит на независимые компоненты (ArtistCard, ErrorPage и пр) для удобства поддержки и масштабирования.
